DS Camera — A Camera App for Direct Sensor Modified Phones

When the original smartphone lens is removed and replaced with an external lens (such as a TV lens, mirrorless lens, or DSLR lens), the captured image often shows distorted or uneven colors. This happens because the phone’s built-in Lens Shading Correction (LSC) is still calibrated for the original factory lens, causing a mismatch with the new optical setup.

DS Camera is designed to minimize this issue by applying an inverted flatfield JPG as a LUT (Lookup Table) correction layer. This compensation helps restore more balanced colors and reduces shading artifacts caused by the lens mismatch.

The app is specifically intended for direct sensor modified phones and experimental external lens setups.

[INDONESIAN TRANSLATION]:
DS Camera — Aplikasi Kamera untuk Ponsel Modifikasi Direct Sensor

Ketika lensa bawaan smartphone dilepas dan diganti menggunakan lensa eksternal (seperti lensa TV, mirrorless, atau DSLR), hasil foto sering mengalami distorsi warna atau ketidakseimbangan pencahayaan. Hal ini terjadi karena sistem Lens Shading Correction (LSC) bawaan ponsel masih dikalibrasi untuk lensa original pabrik, sehingga tidak sesuai dengan konfigurasi lensa yang baru.

DS Camera dirancang untuk meminimalkan masalah tersebut dengan menerapkan file flatfield JPG terbalik (inverted flatfield) sebagai layer LUT (Lookup Table) koreksi. Metode ini membantu menyeimbangkan kembali warna dan mengurangi artefak shading akibat ketidaksesuaian lensa.

Aplikasi ini secara khusus ditujukan untuk ponsel direct sensor mod dan eksperimen penggunaan lensa eksternal.
